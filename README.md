# spark-movie-recommender
Movie recommender using Spark MLlib. 

It uses MovieLens sample data to train the model and do the prediction on test data.

**Build**

```mvn clean install```

**Execution**

```java -jar target/spark-recommender-1.0-SNAPSHOT.jar```
